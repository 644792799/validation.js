Input Object

Input objects are used to bind validation methods and their parameters to each form input. They are designed to be readable and flexible.

The 'key' is the ID of the form input. For inputs to validate must have an ID.

Array literal

An array literal is contained within square brackets ([]) with each value separated by a comma. Values can be any type of data. 

['arrays', 'are', 'easy' 'to', 'understand']
