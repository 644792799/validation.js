Installation

Include the validation.js Javascript file (minified version recommended for production use) within your page and instantiate a new Validation object using the Validate() constructor method which takes 3 arguments: 

var check_this = Validate('my_form', [ methods ], { options });
Form ID

The ID string of your form. <form id="my_form" method="post" action="">

Validation methods

An array of input objects. An array literal is contained within square brackets ([]) and each object is separated by a comma. Find out how to write an input object.

Options

Default options can be overwritten for each Validation.js instance by passing in a new object. An object literal is contained within curly braces ({}) and written in "key: value" format. Find out what options are available.
