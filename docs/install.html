Installation

Include the validation.js Javascript file (minified version recommended for production use) within your page and instantiate a new Validation object using the Validate() constructor method which takes 3 arguments: 

var check_this = Validate('my_form', {validation methods}, {options});
Form ID

The ID string of your form. <form id="my_form" method="post" action="">

Validation methods

Each form input requiring validation is passed into the constructor within an object literal where the key is the name of the input to be. The value is also an object literal and contains the arguments to test the inputs value against. Find out how to write an input object.

Options

Default options can be overwritten for each Validation.js instance by passing in an object literal. Find out what options are available.

Object literal

An object literal is contained within curly braces ({}) and written in "key: value" pairs separated by a comma. Keys must be quoted if they contain hyphens or if they are a global object namesake such as "date" (See a list of global objects). Values can be any type of data. 

{
	object: literals,
	are: 'simple constructs',
	that: 'encapsulate associated data',
	'such-as': [function(), [array], 'strings']
}
